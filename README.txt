This python scripts can be used to generate a whole parts of fire technical documentation.

Folder Fire_Documentation_Generation contains:
	- scripts to calculate fire loads in fire sections
	- scripts to gerarate various tables needed in fire technical documentation
		- table of all fire sections with proper data
		- table comparing actual and allowed size of fire sections
		- fire resistance of construction - still on progress
		- table quantifying number of fire extinguishers
		- appendix - input for the calculation

Folder Odstupy can be used to calculate fire safety distance (fire separation).
	- script writes a part of fire technical documentation
	- script draw the fire separation lines in dxf format


These scripts can be used only in Czech republic as the calculations are done according to czech technical standards
